Как показали опыты уважаемого модмейкера Kurpulio, частицы можно легко создавать полностью в нифскопе!
Быстро и эффективно.
А в сочетании с МВСЕ плагином livemesh edit script - можно сразу просматривать все изменения прямо в игре, без перезагрузок и прочего.
Этот скрипт можно найти в архивах по ссылкам ниже.
 
Его видео руководство по работе с частицами, только средствами нифскопа, можно найти здесь:
https://youtu.be/oZRVoQ8-T74
Язык английский, но в целом можно смотреть и без звука.
Скромные 8 часов (:) ролика покажут все тонкости работы с частицами, без использования 3д редакторов!)))
Рекомендуется скачать файл на диск!
*используя подходящие для этого внешние сервисы.
