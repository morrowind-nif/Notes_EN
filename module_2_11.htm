Chapter 9. Attached Files

Description of content in Additional Files folder
(placed in one archive with the Notes)
Attached_Files\Example_NIFs\
 
These are NIF files for many objects listed in Supported By Engine page. See here which
This is a stripped-down version from the "Videos and examples" folder, which also contain 3dsmax files, and short videos.
("Videos and examples" is uploaded as separate folder because of file size)
 
 
Attached_Files\Nifskope
Different Nifskope versions. Recommended Nifskope v2.dev7.
Nifskope v1.13 also good, free choice, there are only small pros/cons (see here what).
Nifskope v2.dev8 may be good to render other game's newer NIF version files if you want to datamine them.
 
Nifskope
 
 
Nifskope(2.0.7_FOR_MW_ONLY)
 
NifSkope 2.0.dev7 with corrected Nif.xml file
Works only with files for Morrowind!
Removed all other entries for optimization purposes.
(https://github.com/niftools/nifskope/releases)
Nifskope(2.0.8_native_original_nif.xml)
 
NifSkope 2.0.dev8 NifSkope_2_0_2020-07-09-x64.7z
The original nifscope archive of this version with the original nif.xml file.
(https://github.com/hexabits/nifskope/releases/tag/v2.0.dev8)
Nifskope(installers)(1.1.3&1.2a)
 
- 1.1.3 and 1.2a versions of Nifskope. Installer and archive.
Nif.xml is original and will require editing or replacement.
Nifskope_1.1.3(portable)
 
Portable assembly of nifscope 1.1.3
Cons: May not work on some os(win10)
Pros: -Does not modify Windows registry.
- you can easily roll back to the original nif.xml
-Customized interface (black-orange style) (which can be seen in the screenshots in the notes)
 
 
Nif_xml
 
 
Nifskope_(XML_file_Only)(for_1.1._1.2versions)
 
Modified NIF.xml files for Nifskope v1.1.3 and 1.2a
 
1.1.3_for_MW_nifs_ONLY!
 
Modified NIF.xml files for Nifskope v1.13
Only able to open Morrowind Netimmerse 4.0.0.2 NIF files!
 
1.1.3_with_new_edits(can_open_other_version)
 
Modified NIF.xml files for Nifskope v1.13
Able to open other NIF file versions; so also contain unneeded entries for MW Nifs.
 
2.0.7
 
Modified NIF.xml files for Nifskope v2 dev7. May not work for v2 dev8
Only able to open Morrowind Netimmerse 4.0.0.2 NIF files!
 
 
 
Additional_Files\Plugins
Various plugins for 3D MAX. 3,4,5 and 2009 versions.
A plugin for Blender version 0.8.
Plugins for Gimp and Photoshop, ie. to open DDS texture files.
 
DDS plugins
 
 
gimp_dds_plugin
 
The ability to open edit DDS texture files in this 2d editor.
 
photoshop_dds_plugin
 
Plugin for photoshop to be able to open and save .dds texture files also edit mipmaps.
 dds_dxt_tools_v5.11 - for older Photoshop versions (like Photoshop 7 from 2002)
 adobedds_v0.6_win - for newer Photoshop versions (like Photoshop CC from 2015

3DS Max
 
Plugins for 3ds MAX (different versions 3, 4.2, 5.1, 2009) and some others.

TesExporter
 
 
MAX_3
 
- Version for 3ds max 3.1
MAX_4(5)

ExtraFiles
 
 
StdPlugins
 
Morpher.dlm file from MAX 4.2.
Can be set in MAX 5.1 to get rid of CTD when exporting morphs via the TES and FFE exporters.
Optional file, install as desired.
Other_plugins
 
 
animation bakers
 
Animation baking scripts. They will allow you to export complex animations to a nif file.
Convert animations to a regular Keyframe, which can be easily exported to nif files as NiKeyframeController.
MAX 2009 mostly, maybe some of the plugins will work under MAX 5.1 as well.
 
DDS plugins for MAX 3-4
 
The plugin for MAX 3-4.2 allows you to use DDS files.Because By default, these versions of MAX do not handle DDS texture files.
 
de_triangulator for MAX 5-2009
 
A plugin that allows you to remove extra faces on polygons after importing models in MAX.
 
Detach
 
Plugins that allow you to quickly separate (break) an object into separate elements.
 
Scene manager(MAX4-2009)
 
Addition (replacement) to the standard scene manager.Allows you to more conveniently manage the selection of objects.
 
Turn_To_Instance
 
Converts copies of an object to its Instance, i.e. in a nif file allows you to reduce the number of unique shapedata, which will positively affect the file size.
 
 
Attached_Files\LiveMeshEdit
Tool allows Morrowind.exe or OpenMW.exe to reload NIF files when you save them in Nifskope. See here
 
LiveMeshEdit
 
 
MWSE-LiveMeshEdit
 
Use this if you're using Morrowind.exe. Requires MWSE+mgexe+mcp. Can only target 1 nif. Can spawn NIf in front of you
OpenMW-LiveMeshEdit
 
Use this if you're using OpenMW.exe. No need to target 1 nif: any NIF will be reloaded when you save over it.
This OpenMw version is bugged for Skinned NIfs
 
 
Attached_Files\ScreenshotCreator
A small utility that allows you to quickly create screenshots.Both as a file to memory and a file to disk. Ru\Eng.
With its help, you can quickly collect blanks for creating icons in the editor, for example.
 
Attached_Files\SceneImmerse Viewer
Official NIF file viewer! See here. Can show many things that Nifskope can't render.
