How to use the Notes

Few instructions for using this guidebook
•Set up the tools
 
Install 3dsMax or Blender
to create/edit 3d models or animations
 - Get NIF exporter plugins
to be able to export your work into .NIF file format
Install Photoshop or Gimp
to create/edit textures
 - Get DDS exporter plugins
to be able to export textures in .dds format
Install Nifskope 2. dev7
to create/edit Morrowind NIF files that store models/animations/effects
 - Get custom nif.xml file
to optimise Nifskope for Morrowind NIF editing.
Try Livemeshedit (mwse/openmw)
to be able to instantly see how your NIF file works ingame while editing
