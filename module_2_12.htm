Chapter 10. Videos and examples

Videos_and_examples_/@How_To - Main folder: Nif files, 3dsmax 5.1 files, videos.
 How to make each NIF objects with Nifskope or 3dsMax.
    Example:
        AvoidNode
         MX_AvoidNode.flv - now to make AvoidNode with 3ds MAX.
         NF_AvoidNode.flv - how to make AvoidNode with Nifskope. First way.
         NF_AvoidNode2.flv - how to make AvoidNode with Nifskope. Second way.
         AvoidNode.NIF - nif with AvoidNode.
         AvoidNode.max - max file with avoidNode. Open with 3d Max and see.
 
  File names:
        NF_ - vids how to do something via Nifskope.
        MX_ - vids how to do something via 3d MAX.
        _edit - how to edit some values.
        _fix - how to fix something.
        _add - how to add something.
 
Videos_and_examples_/other folders
 
@Textures - folder with textures used for the example Nif and 3ds Max files.
Photoshop - folder with vids, how to make something via Photoshop.
MAX_only - folder with vids, how to make something via MAX.
PIC_Blender_CS - few vids about Blender's  possibilities.
Test_meshes - some nif files. And blank file for 3d MAX (5) with character's skeleton. Good to attach new animations.
                     - animation for cuirass with few constraints using. Nif files.
